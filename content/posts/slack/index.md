## Slackを少し使ってみる

### チャンネルの確認
各言語ごとにチャンネルが用意されているので質問内容に応じて使い分けてください。

### メッセージの投稿
```
// テキストメッセージ
hello

// 絵文字の投稿
hello :smile:

// 引用（1行）
>hello

// 引用（複数行）
>>>hello
world
```

### メッセージへのリアクション

リアクションのボタンをクリック
